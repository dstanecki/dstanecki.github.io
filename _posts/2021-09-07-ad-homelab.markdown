This homelab utilizes VirtualBox to connect a Windows 2019 Server Domain Controller to a Windows 10 client machine. I followed [Josh Makador's tutorial](https://www.youtube.com/watch?v=MHsI8hJmggI) on YouTube to set this up. He also provides a randomly generated list of ~1,000 users to add via PowerShell.<!--break-->
